# OnlineStore
CRUD приложение для "общажного магазина" с каталогами и товарами внутри. Для рядовго пользователя никакого функционала, кроме просмотра ассортимента, не предусмотрено. Все CRUD операции производятся в панели администратора. Также, к аккаунту администратора можно привязать почту, и после её подтверждения использовать для сброса пароля.
